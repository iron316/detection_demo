# detection_demo

## environ
- python version : 3.7.10
    - poetryを使ってます(requirements.txtからも呼べます)
    - souceからライブラリを読むため、poetry経由していないのもあります
- CUDA version : 10.1

## how to run
```
$ make setup-dataset
$ make setup-python-poetry # or setup-python-requirements
$ make run-notebook # work on 8866 port
```
